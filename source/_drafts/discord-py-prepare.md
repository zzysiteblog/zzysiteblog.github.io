# 准备材料
1. Python 3
2. Windows 7 及以上版本（如是Python 3.9及以上版本是Windows 10） / Mac OS / Linux 开发环境
3. Visual Studio Code

# 安装discord.py库

1. 打开PowerShell，输入`pip install discord.py` 。

2. 等待自动安装

> 注意：如没有去国外工具，请百度搜索如何更改pip镜像源为清华大学镜像源
